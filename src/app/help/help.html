Meditation+ is a web application and online community for Sirimangalo International. It provides a meditation timer, meditation log, a chat room, a forum for asking questions, access to live streamed daily dhamma talks, and scheduling for online meditation courses. The website format is mobile device friendly and can be used on any mobile device with access to the web.

Reporting issues, feedback and ideas

If you encounter any bugs or want to send some feedback please use our GitHub repository for that. Note that if your issue is marked as 'closed' the changes will be in the next release and not instantly pushed to the server.

Information for mobile users

If you want to use this app on your smartphone, you can use the Add to Homescreen feature in your mobile browser (Chrome on Android, Safari on IOS) to create an app like experience.

Please note that the meditation alarm is currently not working on mobile phones. Therefore you should use a native timer app on your phone until we've found a stable solution for this problem.

Meditation

The Meditation tab allows you to submit a meditation session to the server and view sessions currently in progress. The list of sessions also shows recently completed sessions below active ones. Above the list there is a graph with the numbers 0-23 below it. The numbers represent hours of the day, and the current hour by UTC standard time is highlighted. The bar graphs represent total minutes of meditation per hour (UTC) for the last month, meant to give an idea of when you are most likely to find other meditators online.

Alarm sound

  • Please note that you need to set an alarm sound in your profile first.
  • Please note that this feature might not work properly on mobile phones. Therefore you should use a native timer app on your phone until we've found a stable solution for this problem.

Chat

The Chat tab allows you to interact with other meditators.

Emojis

  • You can use Emojis by either using the emoji-menu (tag_faces) or manually inserting a codename in your message. A list of Emojis with their codes can be found here.

Questions

  • You have the option to mark your message as a question by either beginning the message with 'Q:' or using the Emoji :question: () anywhere inside your text. An admin then can click on your question and mark it as answered.
  • By clicking on you activate the QUESTIONS ONLY MODE. All messages that are not questions will be hidden.

Commitments

The Commit tab allows you to sign up for various commitments.

Profile

Clicking on an users avatar or name will load the Profile page.

Settings

  • You can select the information displayed for your profile by clicking on .
  • In your profile settings you can choose a default sound for your meditations and change your name, password, description and country.
  • In order to change your profile image, you need to register at Gravatar and set a Globally Recognized Avatar for the email adresse you've signed up with.

Appointments

The Appointments/Schedule page allows users to book one-on-one appointments with Ven. Yuttadhammo Bhikkhu for private meditation instruction. The online course runs for approximately 14 weeks and is offered at no charge. You'll find all necessary information on the page itself.

Testimonials

On the Testimonials page you can write down your experiences with your meditation practice. Please notice that your testimonial will be only displayed after it has been reviewed by an admin.

Livestream

A live broadcast or the time for the next one will be displayed on the Livestream page.
