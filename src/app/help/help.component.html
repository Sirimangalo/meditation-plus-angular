Meditation+ is a web app for the online meditation community of Sirimangalo International. It provides a meditation timer and log, a chat interface for discussion and asking questions, live streaming of daily dhamma talks, and an appointment schedule for online meditation courses. The website can be used on any mobile device with access to the internet.

Reporting issues, feedback and ideas

If you encounter a bug or want to send feedback please use our GitHub repository. Note that if your issue gets marked as 'closed' the changes will come in the next release and not instantly to the app.

Information for mobile users

If you want to use this app on your mobile device, you can use the Add to Homescreen feature in your mobile browser (Chrome on Android, Safari on IOS) to create an app-like experience.

Please note that the meditation alarm is currently not working on mobile devices. Please use a native timer app on your phone until we've found a stable solution for this problem.

Meditation

The Meditation tab allows you to post a meditation session to the app and view sessions currently in progress by others. The list of sessions also shows recently completed sessions below the active ones. Above the list there is a graph with the numbers 0-23 below it. The numbers represent hours of the day, and the current hour is highlighted (this is based on UTC standard time if you haven't set your timezone in your profile). The bar graphs represent total minutes of meditation per hour for the last month, meant to give an idea of when you are most likely to find other meditators online.

Alarm sound

  • Please note that you need to set an alarm sound in your profile in order to hear a sound at the end of your meditation.
  • Please note that this feature might not work properly on mobile phones. Therefore you should use a native timer app on your phone until we've found a stable solution for this problem.

Chat

The Chat tab allows you to interact with other meditators.

Emojis

  • You can use Emojis by either using the emoji menu (tag_faces) or by manually inserting a special tag into your message. A list of Emojis with their tags can be found here.

Questions

The Ask tab allows you to ask questions which will be answered in the live broadcast. The interface also gives you the ability to vote for questions you like and view answered and unanswered questions separately. Only an admin can mark unanswered questions as answered.

Commitments

The Commit page allows you to sign up for various commitments.

Profile

Clicking on an user's avatar or name will load the Profile page.

Settings

  • You can change the information displayed for your profile by clicking on .
  • In your profile settings you can choose a default sound for your meditations and change your name, password, description, and country.
  • In order to change your profile image, you need to register at Gravatar and set a Globally Recognized Avatar for the email adresse you've signed up with.

Badge system

  • You're receiving a badge (stars) for meditating 10 consecutive days.
  • Four badges of the same kind will be merged to a badge of the new level.
  • Badge levels:
    • stars = 10 consecutive days
    • stars = stars stars stars stars
    • stars = stars stars stars stars
    • stars = stars stars stars stars

Appointments

The Appointments/Schedule page allows users to book one-on-one appointments with Ven. Yuttadhammo Bhikkhu for private meditation instruction. The online course runs for approximately 14 weeks and is offered at no charge. You'll find all necessary information on the page itself.

Testimonials

On the Testimonials page you can submit your experiences with the meditation technique found in our booklet on how to meditate. Please note that your testimonial will be only displayed after it has been reviewed by an admin.

Livestream

A live broadcast interface or the time for the next live broadcast will be displayed on the Livestream page.
